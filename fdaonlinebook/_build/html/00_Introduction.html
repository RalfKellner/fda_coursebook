Financial Data Analytics#

Welcome to the Financial Data Analytics course. Due to the advancing digitalization, a large amount of available data on companies, products and individuals has been created in the last decades. At the same time, storage and computing capacities of IT systems are increasing. As a result of these developments, methods of data analysis have constantly evolved and allow automated and complex analyses of data. The goal of data analysis is to gain insights through these analyses, which in the best case generate new values. The content of this course is intended to provide you with insights into which aspects are important for modern data analysis and how it fundamentally works. We focus on the empirical analysis of financial data, however, the methods presented also apply to different research fields. More specifically, the following content is part of the course:

  • Data types, data retrieval and data preparation.

  • Descriptive and graphical analysis of data

  • Regression models

  • Model evaluation

  • Dimension reduction

  • Clustering

In the course we will work according to the motto “learning by doing” and evaluate data independently. The course material consists of:

  • This online book for the course

  • Digital worksheets

You are already reading the book and accordingly you have found the access. The notebooks can be accessed using links like these. The course starts first with a small Python introduction. Python is one of the most popular programming languages in the world and is particularly useful for data analysis, as access to cutting-edge data analysis technology and methods is provided by Python. The point is not that you should learn programming using Python entirely as part of the course. Rather, Python is a tool for data analysis. The course is primarily about you understanding the output of the programmed cells and, more importantly, interpreting it appropriately. We will start with descriptive analysis and continue with the analysis of individual target variables. We then move on to dimensionality reduction and clustering techniques, which focus on the interaction of all data. For all content, you will find videos and examples in digital worksheets (notebooks). The notebooks give you the possibility to experience data analysis without the knowledge of a programming language like python. Furthermore, these workbooks are discussed in the course tutorial together with some manual calculations.

This procedure serves as preparation for the course exam. The exam consists of:

  • calculations

  • interpretation of data analysis

  • open text questions

  • selection and choice type questions

Literature to accompany the book#

Should you wish to further deepen the knowledge presented during the course, I can especially recommend the following two books as companion reading:

  • James, G., Witten, D., Hastie, T., Tibshirani, R., Taylor, J. (2023): An Introduction to Statistical Learning with Applications in Python, Springer Verlag.

  • Ruppert, D., Matteson, D.S. (2015): Statistics and Data Analysis for Financial Engineering with R examples, Springer Verlag.
